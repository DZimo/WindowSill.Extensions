# 🪟 WindowSill Extensions

A collection of lightweight productivity tools designed to enhance your desktop workflow.

---

## 🧰 Tools

| Tool | Description |
|------|-------------|
| ⏱️ **Pomodoro Timer** | Stay focused and manage time effectively using the Pomodoro technique |
| 🎨 **Color Picker** | Pick, convert, and manage colors easily |
| 🧮 **Smart Calculator** | Perform quick and intelligent calculations |
| 🎥 **Screen Recorder** | Record your screen or capture screenshots instantly |
| 📅 **Outlook Calendar** | Integration of outlook calendar to get notified about next meetings. |
